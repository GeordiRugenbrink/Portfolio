Terrain Generator

Dit project heb ik gemaakt als onderzoek voor het themasemester Gameplay engineering. Hierbij wilde ik onderzoeken hoe er procedureel een terrein gegenereerd kon worden met procedureel gegenereerde huizen erop. Ik vind persoonlijk dat dit een goed begin is voor het genereren van een terrein dat gebruikt kan worden in een multiplayer shooter/survival game. Voor verder onderzoek kan er nog gekeken worden naar hoe er bomen op het terrein gezet kunnen worden en hoe de huizen overlappende stukken mesh weghalen en/of veranderen. De demo van het project is hier te downloaden, in de demo kan je een nieuw terrein laten genereren waar vervolgens dan de huizen op worden gezet.

A Knight's Bounty

Deze game is gemaakt tijdens de Ludum Dare 41 compo (de 48 uur durende versie van de Ludum Dare game jam). Het thema van de Ludum Dare was "Combineer twee niet samengaande game genres". Het was de bedoeling om de genres Rpg en bullet hell te combineren, maar ik had niet genoeg tijd voor het bullet hell gedeelte. De game is hier te vinden samen met de source code.
