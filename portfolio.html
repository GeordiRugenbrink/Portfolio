Terrain Generator

This is a solo project where I researched how to create a terrain generator that also spawns objects on top of the terrain. This research was done for the theme semester Gameplay Engineering. The major reason for wanting to research this specific thing was to see if this could be used for making a multiplayer shooter/survival game. But this needs far more ressearch to check what makes a map good for a shooter/survival game.

The demo project can be downloaded here, in the demo you can generate a new terrain where objects are spawned on top of the terrain using poisson disc sampling.

A Knight's Bounty

This is a game I made during the Ludum Dare 41 compo (the 48 hour long version of the Ludum Dare game jam). The theme for this Ludum Dare was "Combine 2 incompatible genres". The game was supposed to combine the genres; rpg and bullet hell, but I ran out of time so I left out the bullet hell part.

The game can be found here together with the source code.
